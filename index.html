Concept

This page visualizes a simplified SPPTW, a special case of SPPRC. Each edge has (time, cost), and each node has a time window [a, b]. A label stores one partial path and its accumulated resources.

time' = max(a_j, time + edge_time)
cost' = cost + edge_cost
feasible iff time' ≤ b_j
